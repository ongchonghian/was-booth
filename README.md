# WAS 2026 Booth Planning

**IMPORTANT CONTEXT FOR AGENTS AND COLLABORATORS:**
The content in this repository is strictly for **internal planning** for the **World Aquaculture Singapore 2026 (WAS 2026) booth engagement strategy**.

## What this repo is:
- A trade show engagement deck and planning repository.
- A strategy for how visitors will experience SGFarmApp through mini-games at the booth.
- Concepts designed to attract visitors, teach product insights quickly, and convert them to conversations/leads.

## What this repo is NOT:
- **NOT** a product feature ideation deck.
- **NOT** a recommendation to build new gamification features into the core SGFarmApp.
- **NOT** product backlog items for the engineering team.

All "gameplay" and "gamification" references in this repo describe **booth play flows** for visitors, using standalone demo setups, and do not represent features to be implemented in the core app for farm workers.

## Project Files
- `index.html`: The main interactive infographic for internal alignment, stakeholder buy-in, and booth-floor execution.
- `docs/`: Concept notes, booth flow recommendations, and strategy drafts.
